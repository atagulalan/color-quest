# Color Find Game

A challenging color-finding puzzle game where players must identify the different colored cell in a grid. The game features 100 progressively difficult levels with increasing color similarity and grid complexity.

## Features

- 100 levels with progressive difficulty
- Grid sizes from 2x2 to 7x7
- Seeded random color generation for consistent level experience
- Timer system with audio feedback
- Multiple shape types (box, heart, circle, star)
- Settings for vibration, music, and sound effects
- Responsive design with mobile support

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Game Mechanics

- Early levels (2x2, 3x3) have 10-second timers
- Later levels have 15-second timers
- Clock tick sound plays in the last 5 seconds
- Colors become increasingly similar as levels progress
- Grid size and number of different colors increase with difficulty
